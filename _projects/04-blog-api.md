API RESTful completa para plataforma de blog con autenticación, CRUD de artículos, comentarios y sistema de categorías.

Backend desarrollado en PHP con arquitectura MVC, base de datos MySQL y contenedores Docker para el entorno de desarrollo. Incluye documentación completa con OpenAPI/Swagger.

**Características principales:**

- Arquitectura RESTful con HATEOAS
- Autenticación con tokens JWT
- Rate limiting y caché con Redis
- Tests unitarios y de integración
- CI/CD con GitHub Actions y Docker
