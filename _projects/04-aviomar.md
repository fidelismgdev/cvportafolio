Desarrollo integral (front-end y back-end) de múltiples sitios y plataformas para el grupo de turismo de aventura Aviomar AdvenTours en Cozumel.

**Proyectos desarrollados:**

- [Aviomar AdvenTours](https://www.aviomar.com/) — Rediseño del sitio corporativo
- [The Mayan Cacao Company](http://www.themayancacaocompany.com/) — Optimización de código y booking
- [Mayan Bee Sanctuary](http://mayanbeesanctuary.com.mx/) — Sitio completo con sistema de reservaciones

**Responsabilidades:**

- Rediseño completo del sitio corporativo
- Creación de motor de reservas y panel de administración (ventas, reservaciones, reportes)
- Desarrollo de API REST para tours corporativos
- Back-end, front-end, despliegue y optimización SEO
