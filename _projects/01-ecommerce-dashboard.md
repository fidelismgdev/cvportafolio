Panel de administración para tienda en línea con visualización de datos en tiempo real, gestión de inventario y análisis de ventas.

Desarrollado con React en el frontend y Node.js con Express en el backend, utilizando MongoDB como base de datos. El dashboard incluye gráficos interactivos, filtros avanzados y exportación de reportes en PDF.

**Características principales:**

- Dashboard con métricas en tiempo real
- Gestión completa de productos e inventario
- Sistema de autenticación con JWT
- API RESTful documentada con Swagger
- Responsive design para todos los dispositivos
