Sitio web corporativo de alto rendimiento con animaciones suaves, optimización SEO y score perfecto en Lighthouse.

Diseñado y desarrollado desde cero sin frameworks pesados, priorizando la velocidad de carga y la accesibilidad. Implementa técnicas modernas de CSS como Grid, Custom Properties y animaciones con Intersection Observer.

**Características principales:**

- Score de 100 en Lighthouse (Performance, SEO, Accessibility)
- Animaciones con CSS puro e Intersection Observer API
- Formulario de contacto con validación
- Integración con Google Analytics
- Deploy automatizado con GitHub Actions
