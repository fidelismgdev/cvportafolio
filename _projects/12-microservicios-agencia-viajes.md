Sistema de microservicios en desarrollo para una agencia multinacional de agentes de viajes especializada en tiempos compartidos. Arquitectura distribuida diseñada para escalar operaciones de venta y gestión de membresías a nivel internacional.

**Stack tecnológico:**

- **Backend:** NestJS (Node.js) con arquitectura de microservicios
- **Frontend:** React.js
- **Base de datos:** PostgreSQL
- **Infraestructura:** AWS (cloud)

**Características del sistema:**

- Arquitectura de microservicios desacoplada para alta disponibilidad
- Gestión de agentes de viajes y membresías de tiempo compartido
- Infraestructura en AWS para operaciones multinacionales
- APIs RESTful entre servicios
