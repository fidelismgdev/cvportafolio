Aplicación web para gestión de proyectos y tareas con autenticación, colaboración en tiempo real y notificaciones push.

Construida con Angular y Firebase, aprovechando Firestore para sincronización en tiempo real y Cloud Functions para lógica del servidor. Incluye sistema de tableros estilo Kanban con drag-and-drop.

**Características principales:**

- Tableros Kanban con drag-and-drop
- Sincronización en tiempo real con Firestore
- Autenticación con Google y email
- PWA instalable con soporte offline
- Notificaciones push
